# Yarn

### Install yarn:

```bash
corepack enable #Enable Corepack
yarn set version stable #Install latest version of Yarn
yarn install #Install dependencies
```

### Upgrade dependencies across whole project

```bash
yarn up
```

### Install dependencies:

```bash
yarn install
```

### Start dev server:

```bash
yarn dev
```
